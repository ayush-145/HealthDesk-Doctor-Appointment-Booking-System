# Contributing to HealthDesk

Thank you for your interest in contributing to HealthDesk! 🎉

## How to Contribute

### Reporting Bugs
- Use the [GitHub Issues](https://github.com/ayush-145/HealthDesk/issues) tab
- Include steps to reproduce, expected behavior, and screenshots if possible

### Suggesting Features
- Open an issue with the **Feature Request** label
- Describe the feature and why it would be useful

### Code Contributions

1. Fork the repo and create your branch from `main`
2. Install dependencies for all three modules (frontend, admin, backend)
3. Make your changes and test thoroughly
4. Ensure your code follows the existing style
5. Write a clear commit message
6. Open a Pull Request

### Development Setup

```bash
# Clone your fork
git clone https://github.com/YOUR_USERNAME/HealthDesk.git
cd HealthDesk

# Install all dependencies
cd backend && npm install
cd ../frontend && npm install
cd ../admin && npm install

# Start development servers (3 terminals)
cd backend && npm run server
cd frontend && npm run dev
cd admin && npm run dev
```

## Code of Conduct

Be respectful, constructive, and welcoming. We are all here to learn and build together.
